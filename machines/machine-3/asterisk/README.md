### Configure

`extentions.conf -> /etc/asterisk/extentions.conf`
`sip.conf -> /etc/asterisk/sip.conf`
`search_engine -> /usr/share/asterisk/agi-bin/search_engine`
`sounds/* -> /usr/share/asterisk/sounds`
`dict.txt -> /usr/share/dict.txt`

```bash
chmod +x /usr/share/asterisk/agi-bin/search_engine
chown asterisk /usr/share/asterisk/agi-bin/search_engine
```
